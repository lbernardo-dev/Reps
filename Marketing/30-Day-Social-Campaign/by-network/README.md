- Instagram: usar `IG` (carrusel o imagen) y reutilizar el guion de `TT` como Reel.
- TikTok/Reels: usar `TT`; todas las escenas deben ser grabación o captura del simulador.
- LinkedIn: usar `LI`; conservar saltos de línea y tono reflexivo.
- X: usar `X`; publicar como post único o mini-hilo numerado cuando se indique.
- Facebook: usar `FB`; admite el copy más explicativo y una pregunta final.
